## 计算题
`考察范围：第一、二章`
考虑用一段C语言程序和它编译为MIPS指令的版本，其中array为长度为len的int类型数组，array的首地址已保存在$s0中，len保存已在$s1中，回答下列问题
```C
int f1(int arg){
    if(arg>=0 && arg<len)
        return array[arg];
    else return 0;
}

void f2(int arg){
    for(int i=-arg; i<=arg; i++)
        f1(i);
}
```

```mips
f1:
    addi $sp, $sp, -8
    sw   $ra, 4($sp)
    sw   $a0, 0($sp)
    # (1)
    beq  $t0, $zero, else
    # (2)
    j    rtn
else:
    add  $v0, $zero, $zero
rtn:
    lw   $a0, 0($sp)
    lw   $ra, 4($sp)
    addi $sp, $sp, 8
    jr   $ra

f2:
    addi $sp, $sp, -8
    sw   $ra, 4($sp)
    sw   $a0, 0($sp)
    # (3)
loop:
    jal  f1
    addi $t0, $t0, 1
    slt  $t1, $a0, $t0
    beq  $t1, $zero, loop
    lw   $a0, 0($sp)
    lw   $ra, 4($sp)
    addi $sp, $sp, 8
    jr   $ra
```
